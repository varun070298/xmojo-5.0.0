The following gives the consolidated list of bugs or known issues:
  • Model MBean:
    • "currencyTimeLimit" is not working whatever the value is set.
    • "store" (PersistMBean) is not working properly in case of RequiredModelMBean.
  • Timer Service:
    • "sendPastNotification" does not have any effect either set as true or false.
  • Counter Monitor Service:
    • Rollback fails, i.e., "setModulus" does not have any effect.
  • M-Let Service:
    • If invalid MLet tags are given for MLet, appropriate exceptions are not thrown.
  • MBeanBrowser Tool:
    • MBeanInfo is not refreshed in case of Dynamic MBean.
